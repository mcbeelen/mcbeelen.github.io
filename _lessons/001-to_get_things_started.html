First assignment: Start to keep an online journal about everything you learned

Rationale

  1. Keeping a journal forces you to digest the presented content and summarize it for yourself.
    This reinforces the learning of the content.
  2. It also become a reference for yourself and other to look back at.
  3. It provides a nice starting point to talk about some fundamentals of the web like HTML, browers and hosting

Approach

I did not want to teach them just some programming language, I wanted them to get used to some best practices right a way. So the first assignment had to be about getting up and running with some kind of Version Control an of course GIT was the most logical option.

This allowed us to use Github Pages for hosting the online journals.

Part of this lesson

  1. Find a professional username, which isn't used on github.com and create an account for yourself
    Availability of the same username of Twitter was recommended.
  2. Install Sourcetree for some basic GIT-
  3. Create a proper repo of your Personal Github Page and check it out
  4. Write a 'Hello World' into index.html
  5. Git command: Commit, Push
  6. View the results
