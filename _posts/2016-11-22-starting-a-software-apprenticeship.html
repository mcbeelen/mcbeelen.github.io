Introduction

While talking with Simone Kamer about a career switch I started to think about things to teach somebody new to the profession of a Software Engineer.
She was interested to learn and I'm enthousiastic about teaching and coaching, so a Software Apprenticeship was the logical next step.

When the Software Apprenticeship really started I wondered if my brother-in-law Sjoerd van Houten might be interested as well. One text message later the 'class of 2016' had it's second student.

First assignment: Version Control

I did not want to teach them just some programming language, I wanted them to get used to some best practices right a way. So the first assignment had to be about getting up and running with some kind of Version Control an of course GIT was the most logical option.

The assignment was:

  1. Find a professional username, which isn't used on github.com and create an account for yourself
    Availability of the same username of Twitter was recommended.
  2. Start to keep a journal on our individual Github Pages
